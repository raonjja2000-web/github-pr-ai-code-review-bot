# GitHub PR AI Code Review Bot

An AI-powered GitHub Action that automatically reviews pull requests using an LLM API.

## Features

- Automatically triggers when a Pull Request is opened or updated
- Analyzes changed code
- Generates AI-based review comments
- Uses GitHub Actions for CI automation

## How It Works

1. A Pull Request is created.
2. GitHub Actions runs the workflow.
3. A Python script sends the changed code to an AI API.
4. The AI generates review comments and posts them to the Pull Request.

## Tech Stack

- Python
- GitHub Actions
- GitHub REST API
- OpenAI / Groq API
